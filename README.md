# Ignite DEX Documentation

Ignite DEX is a high-performance, permissionless decentralized exchange designed to combine CEX-grade trading performance with non-custodial settlement and native multi-chain execution.

## Overview

Core properties:

- Hybrid off-chain matching with on-chain settlement
- Spot and perpetual futures trading
- Multi-chain support across Lithosphere, Base, Arbitrum, and BNB Chain
- Unified portfolio and shared margin abstractions
- AI-native trading systems and autonomous agents
- IGNT as the ecosystem fee, incentive, and governance token

This documentation set is structured for traders, integrators, market makers, and protocol engineers.

## Documentation map

- **Getting Started** explains the product, chains, and terminology
- **Architecture** explains how Ignite is built
- **Trading** explains market behavior, margin, fees, and liquidations
- **APIs & SDKs** covers REST, WebSocket, binary transport, and SDK usage
- **Smart Contracts** describes the on-chain system and includes ABIs
- **AI Layer** documents autonomous agent infrastructure
- **Tokenomics** describes IGNT utility and economic flows
- **Operations** covers observability, security, client platforms, and roadmap
