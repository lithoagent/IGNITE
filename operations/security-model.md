# Security Model

Security spans contracts, infrastructure, operations, and user protections.

## Contract security

- external audits before mainnet
- multisig administration
- timelocked upgrades
- emergency pause

## User security

- non-custodial wallets
- WalletConnect support
- hardware wallet compatibility

## Protocol security

- circuit breakers
- oracle redundancy
- margin and liquidation enforcement
- replay protection in bridging
