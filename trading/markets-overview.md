# Markets Overview

Ignite supports a professional market structure centered on a central limit order book.

## Product categories

- Spot markets
- Perpetual futures
- Future options markets
- Permissionless launch and listing markets

## Core execution behaviors

- limit, market, IOC, FOK, post-only order types
- maker/taker fee model
- cross-chain liquidity awareness
- deterministic match sequencing

## Market metadata

Each market exposes:

- symbol and base/quote asset
- tick size and lot size
- leverage cap
- funding interval
- risk tier schedule
- supported collateral configuration
